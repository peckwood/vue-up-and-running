Adding new properties to an object

username: {{ formData.username }}

real name: {{ formData.realname }}


While the formData.username property will be reactive and respond to changes, the formData.name property will not

Try with
app.formData.realname = 'Black';


Finally, Vue provides a function called Vue.set() that you can use to set reactive properties: Vue.set(vm.formData, 'name', 'Some User');
When inside a component, this function is also available as this.$set

Try with
app.formData.nickname = 'Claymore';

nick name: {{ formData.nickname }}
